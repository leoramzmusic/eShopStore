# eShopStore

**Descripción breve:** Ecommerce modular y administrable para cualquier negocio (abarrotes, ropa, tecnología, etc.).

**Objetivo:** Permitir a los usuarios crear su tienda, administrar productos, categorías, promociones e inventario, y ofrecer compras físicas u online.

## Stack tecnológico
- **Backend:** Python (Django + Django REST Framework)
- **Frontend:** Angular
- **Base de datos:** PostgreSQL
- **Infraestructura:** Docker + Kubernetes + Jenkins + Terraform

## Funcionalidades principales
- Gestión de catálogo (CRUD productos, categorías, variantes, buscador).
- Carrito de compras y pedidos.
- CRM de clientes y modo tendero (ventas físicas).
- Pasarelas de pago (Stripe, PayPal, Conekta).
- Métodos de envío configurables.
- CMS para contenido y marketing.
- Dashboard de métricas y SEO.
- Roles y permisos granulares.

## Roadmap de Desarrollo
- **Fase 1 – MVP:** Catálogo, carrito, pedidos básicos, inventario, modo tendero.
- **Fase 2 – Expansión:** CMS, promociones, métodos de envío, dashboard.
- **Fase 3 – Optimización:** SEO avanzado, Kubernetes + monitoreo, integraciones.
